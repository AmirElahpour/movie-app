# 🎬 Movie App

A simple movie browsing website built with **React**, **Vite**, and **TailwindCSS**.

## 🚀 Tech Stack
- React (Vite)
- TailwindCSS
- TMDB API

## ✨ Features
- Browse trending and popular movies
- Responsive UI (mobile-friendly)
- Search movies by name
- Trending movies
- Dynamic movie cards
